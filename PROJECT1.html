FULL STACK

Full stack development is the process of designing, creating, testing, and deploying a complete web application from start to finish. It involves working with various technologies and tools, including front-end web development, back-end web development, and database development.

Front end

A front end developer is responsible for bringing a digital productâ€™s user interface design to life, converting web-design files into the relevant code (HTML, CSS, etc.). Everything that a user clicks, reads, views, or watches have been put in place by a front-end developer.

Back end

Back-end developers ensure the website performs correctly, focusing on databases, back-end logic, application programming interface (APIs), architecture, and servers. They use code that helps browsers communicate with databases, store, understand, and delete data

data base administration

A database administrator, or DBA, is responsible for maintaining, securing, and operating databases and also ensures that data is correctly stored and retrieved. In addition, DBAs often work with developers to design and implement new features and troubleshoot any issues.
